# Modern App Style v65

Ziel: Die Bestell-Web-App auf der Subdomain `bestellen.kristall-doener-wendelstein.de` soll nicht wie eine alte Webseite wirken, sondern wie eine schnelle, moderne Lieferdienst-App.

## Design-Ziele

- Mobile-first
- Cleanes App-Gefühl
- Junge Zielgruppe
- Helle Oberfläche mit warmen Orange-Akzenten
- Runde Kacheln und Karten
- Große Produktbilder
- Bottom-Navigation auf Handy
- Klarer Warenkorb und Checkout
- Kein Webseiten-Footer, keine normale Webseite-Navigation

## Technische Änderung

Das Paket `kristall_bestellen_v65_modern_app_style.zip` ersetzt nur:

```text
assets/css/kd_customer_ui.css
kd_app/bootstrap.php
```

Es gibt keine Datenbankänderung und keine Änderung an Produktdaten.

## Upload-Ziel

```text
/kristall-doener-wendelstein/App/bestellen/
```

Danach im Browser `Strg + F5` drücken.
